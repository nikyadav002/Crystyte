# crystyte

3D crystal structure viewer in the browser. No server, no login.

Supports **POSCAR / CONTCAR / .vasp**, **CIF**, and **XYZ** files.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Pushes to `main` auto-deploy to GitHub Pages via the included Actions workflow.
